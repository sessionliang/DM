ASP.NET Boilerplate is distributed on nuget. Here, a list of all packages.

Framework

Abp

Core package of ASP.NET Boilerplate system. All other packages depends on this package.

Abp.Web

Provides common classes and resources both of MVC and Web API use.

Abp.Web.Resources

Resources (js,css... files) for web layer.

Abp.Web.Mvc

Provides ASP.NET MVC specific classes and resources.

Abp.Web.Api

Provides ASP.NET Web API specific classes and resources.

Abp.EntityFramework

Provides base classes and adapters to work with EntityFramework.

Abp.NHibernate

Provides base classes and adapters to work with NHibernate.

Abp.FluentMigrator

Provides some simple extension methods to use ABP with FluentMigrator.

Abp.AutoMapper

Provides some simple attributes and extension methods to make AutoMapper easier to use.

Test base

Abp.TestBase

Provides base classes to create unit and integration tests for ABP based projects easier.
